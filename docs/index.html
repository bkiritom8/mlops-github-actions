Quick Start

Get started with the MLOps pipeline

Clone and Setup 
# Clone the repository
git clone https://github.com/bkiritom8/mlops-github-actions.git
cd mlops-github-actions

# Install dependencies
pip install -r requirements.txt
pip install -e .

# Run the training pipeline
python scripts/run_training.py --dataset classification --model random_forest
Run Tests 
# Run all tests
pytest tests/ -v

# Run with coverage
pytest tests/ --cov=src --cov-report=html
Trigger Training via GitHub Actions 
# Go to Actions tab in GitHub repository
# Select "ML Training Pipeline" workflow
# Click "Run workflow"
# Select dataset and model type
# Click "Run workflow" button
